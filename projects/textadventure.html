Python Story: Text Adventure

The development of my python text adventure game utilized open source software called repl.it which allowed me to write my code from scratch, which eventually lead to the functional game created by my partner Gabe and I.

We had difficulties when it came to creating the variable and setting the state in order to access this variable when calling it with our loop.

We solved this issue by setting the state to global in order to be accessible by other variables. Consequently, we had a loop set that checked if it was set to true then it would execute our code.This was a collaborative project, and I had one partner, Gabe.
